# Online Library System

# Project Description
->This is an Online library system built using React(Vite).
-> This allows user to Browse books, Add a book, displays the details of the book,and displays the books according to the category.

# How to run the project
-> First clone the project
-> Install node-modules using "npm install"
->Run the project using "npm run dev"

# Features added
->Home page displays all the popular books and a welcome message.
->When we click on view details it displays all the details of the book.
->when we click on any category, it shows the books of that particular category only.
->We can add the books in the dummy data.
->Added error message, when we route the different url.

# Technologies included
->React
->React-Redux
->React-toolkit
->Hooks
->HTML
->CSS
->Javascript
